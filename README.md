# 使い方
"四字熟語を作る"ボタンをクリックするだけで四字熟語が生成されます。  
生成された四字熟語の使い方、読み方はユーザ次第です。

# 技術スタック
- TypeScript（React）
- TailwindCSS
- Vercel（ホスティング）
